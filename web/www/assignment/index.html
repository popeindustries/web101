web 101 - assignment

Your assignment for this course is to build your very own portfolio/gallery of work. It doesn't matter what that work is (design, coding, carpentry, baking, whatever), as long as you and your work are the main focus.

    Some rules:

  • Mobile first: the site should work well on mobile devices
  • Client-side: this course covers front-end development, so no php/wordpress/backend tech
  • Dynamic: the use of AJAX and other dynamic approaches is encouraged
  • Useable on IE: the site should work in IE (but it doesn't need to work the same everywhere)
